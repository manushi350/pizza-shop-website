🍕 Pizza Pie – Pizza Shop Website

Pizza Pie is a responsive pizza shop website where users can explore delicious pizzas, add items to a cart, and place orders online. The website provides a smooth and interactive food ordering experience with a modern design and simple user interface.

📌 Project Overview

This project is a frontend pizza ordering website developed using HTML, CSS, and JavaScript. It allows customers to browse pizza menus, view prices, add items to a cart, and place orders easily.

The website is designed to be responsive and user-friendly, making it accessible on both desktop and mobile devices.

✨ Features

🍕 View different pizza menu items

🛒 Add items to shopping cart

❌ Remove items from cart

📊 Cart shows total items and price

🌙 Dark mode toggle

📱 Fully responsive design

📝 Feedback form for users

🚚 Online order confirmation

🎨 Smooth animations using ScrollReveal

🛠️ Technologies Used

HTML5 – Website structure

CSS3 – Styling and responsive layout

JavaScript – Cart functionality and interactivity

Boxicons – Icons for UI

ScrollReveal.js – Animation effects
